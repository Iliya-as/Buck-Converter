# Buck Converter with LMR16030

A high-efficiency step-down (buck) converter designed with **LMR16030** switching regulator from Texas Instruments.

## Specifications

| Parameter | Value |
|-----------|-------|
| Input Voltage | Adjustable via step-down transformer |
| Output Voltage | Adjustable |
| Max Output Current | 3A |
| Switching Frequency | 500 kHz |
| Switching IC | LMR16030 (TI) |
| Protection | Input fuse & Output fuse |

## Key Features

- Adjustable output voltage
- 3A maximum output current
- 500 kHz switching frequency (good balance between efficiency and component size)
- Input and output overcurrent protection (fuses)
- Compact and reliable PCB layout
